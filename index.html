Insignias de GitHub

Corazón en la manga (Heart On Your Sleeve)

Reacciona a algo en GitHub con un emoji ❤️.

Open Sourcerer

El usuario tiene pull requests combinados en múltiples repositorios públicos.

Starstruck

Crear un repositorio y obtener 16 estrellas o más.

Quickdraw

Cerrar una incidencia o una pull request dentro de los 5 minutos posteriores a la apertura.

Par extraordinario (Pair Extraordinaire)

Ser coautor en una pull request de dos o más contribuyentes.

Pull Shark

2 pull requests fusionadas o más.

Galaxy Brain

2 respuestas aceptadas o más.

YOLO

Fusionar al menos una pull request sin revisión de código.

Patrocinador público (Public Sponsor)

Patrocinar repositorios de código abierto a través de GitHub Sponsors.

Colaborador de Mars 2020 (Mars 2020 Contributor)

Código contribuido a los repositorios utilizados en la Mars 2020 Helicopter Mission. Ahora no se puede obtener.

Programa de archivo GitHub 2020 (Arctic Code Vault Contributor)

Código contribuido a un repositorio en el 2020 GitHub Archive Program. Ahora no se puede obtener.
